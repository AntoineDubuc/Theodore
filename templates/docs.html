Quick Start

Get up and running with Sales Scout in under 5 minutes.

1

Sign Up & Get API Key

Create your account and get your API key from the dashboard.

curl -X POST https://api.salesscout.ai/auth/register \ -H "Content-Type: application/json" \ -d '{ "email": "your@email.com", "company": "Your Company", "full_name": "Your Name" }'
2

Research Your First Company

Start extracting intelligence from any company website.

curl -X POST https://api.salesscout.ai/v2/research/start \ -H "Authorization: Bearer YOUR_API_KEY" \ -H "Content-Type: application/json" \ -d '{ "company_name": "Stripe", "website": "https://stripe.com" }'
3

Connect to HubSpot

Enable automatic CRM sync with 77 enriched fields.

🔗 Connect HubSpot

Authentication

Sales Scout uses API keys for authentication. Include your API key in the Authorization header.

Base URL

https://api.salesscout.ai

Authentication Header

Authorization: Bearer YOUR_API_KEY
🔒 Security Note: Never expose your API key in client-side code. Always make requests from your backend server.

Company Research API

Extract comprehensive intelligence from any company using our 4-phase AI pipeline.

POST /v2/research/start

Start AI-powered research for a company

Request Body

{ "company_name": "Stripe", "website": "https://stripe.com", "config": { "extract_leadership": true, "extract_tech_stack": true, "extract_funding": true } }

Response

{ "job_id": "job_abc123", "status": "started", "estimated_duration": "30-45 seconds", "progress_url": "/v2/research/progress/job_abc123" }
GET /v2/research/progress/{job_id}

Check research progress and get results

Response (Complete)

{ "status": "completed", "progress": 100, "company_data": { "name": "Stripe", "industry": "FinTech - Payment Processing", "business_model": "B2B SaaS", "company_stage": "Enterprise", "tech_stack": ["React", "Ruby", "Go", "AWS"], "leadership": [ { "name": "Patrick Collison", "title": "CEO", "linkedin": "https://linkedin.com/in/patrickcollison" } ], "funding_status": "Private", "employee_count_range": "1000-5000", // ... 70+ more fields } }

HubSpot Integration

Automatically sync 77 company intelligence fields to your HubSpot CRM.

Setup Steps

  1. Connect Your HubSpot Account

    Navigate to Settings → Integrations → HubSpot and click "Connect"

  2. Configure Field Mapping

    Our system automatically maps 77 intelligence fields to HubSpot properties

  3. Enable Auto-Sync

    Turn on automatic sync to update your CRM whenever you research a company

Data Schema

Complete reference for all 77 fields returned by Sales Scout intelligence extraction.

🏢 Company Basics

  • name - Company name
  • website - Primary website URL
  • industry - Industry classification
  • description - Company description
  • founded_year - Year founded

💼 Business Intelligence

  • business_model - B2B/B2C/Marketplace
  • company_stage - Startup/Growth/Enterprise
  • revenue_model - How they make money
  • target_market - Primary customers
  • competitive_advantages - Key differentiators

👥 People & Culture

  • leadership - Executive team
  • employee_count_range - Company size
  • hiring_signals - Recent job postings
  • company_culture - Values and culture
  • remote_policy - Work arrangement

🔧 Technology

  • tech_stack - Technologies used
  • integrations - Third-party tools
  • security_features - Security measures
  • api_availability - API offerings
  • platform_type - Web/Mobile/Both

Error Handling

Understanding and handling API errors effectively.

HTTP Status Codes

200 Success - Request completed successfully
400 Bad Request - Invalid parameters
401 Unauthorized - Invalid API key
429 Rate Limit Exceeded
500 Server Error - Contact support

Error Response Format

{ "error": { "code": "INVALID_COMPANY", "message": "Company website could not be accessed", "details": "The provided website URL returned a 404 error" } }
